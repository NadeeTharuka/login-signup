
# ReactJS Sign In & Sign Up Form
This project is a simple Sign In and Sign Up form built using React JS, providing a user-friendly interface for user authentication.

## Features
- Sign In: Allows existing users to log in with their credentials.
- Sign Up: Enables new users to register by providing necessary information.
- Form Validation: Includes basic form validation for user inputs.
- Responsive Design: Adapts to different screen sizes for a seamless user experience.

## Technologies Used
- React JS: Frontend library for building user interfaces.
- HTML: Structure of the forms.
- CSS: Styling and layout of the forms.
- JavaScript: Logic for handling form submission and validation.

## Getting Started
- Clone the repository:

git clone https://github.com/Nadee-Tharuka/react-signin-signup-form.git

- Navigate to the project directory:

cd react-signin-signup-form

- Install dependencies:

npm install

- Start the development server:

npm start
Open http://localhost:3000 in your web browser to view the application.

## Usage
- Navigate between the Sign In and Sign Up forms using the provided links.
- Enter the required information and submit the forms to log in or register.
- Customize the forms and validation logic as needed for your application.
